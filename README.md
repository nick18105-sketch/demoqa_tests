# DemoQA UI Tests

## 🔧 Stack:
- Java
- Selenide
- JUnit 5
- Gradle

## 📌 Description:
Автоматизированный UI тест формы с сайта demoqa.com

## ✅ What is covered:
- Заполнение формы
- Работа с input, radio, checkbox
- Datepicker (дата рождения)
- Autocomplete (Subjects)
- React-select (State/City)
- Assertions (проверка результата)

## ▶️ Run tests:
